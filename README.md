## How to Use

### OpenAI API Key

**Linux**

```sh
# Register (Current Shell Only)
$ export OPENAI_API_KEY="<YOUR_KEY>"

# Check
$ echo $OPENAI_API_KEY
```

**Windows (PowerShell)**

```sh
# Register (Current Shell Only)
$ $env:OPENAI_API_KEY="<YOUR_KEY>"

# Check
$ $env:OPENAI_API_KEY
```
